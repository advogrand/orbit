# ORBIT House Landing

Premium scrollytelling landing page for ORBIT House.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Deployment

The repository includes a GitHub Pages workflow. On each push to `main`, GitHub Actions builds the site and publishes the `dist` folder.

If Pages is not enabled yet, open the repository settings on GitHub and set Pages source to GitHub Actions.
